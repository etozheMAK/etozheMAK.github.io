Программа Hello, world! приведена ещё в первом издании книги «Язык программирования Си» Кернигана и Ритчи:

#include <stdio.h>
 
int main(void) // Не принимает аргументы
{  
    printf("Hello, world!\n");   
    return 0; // Удачное завершение программы
}
